## How to fill these in

1. Create (or use) one Google Doc per stack. Put **all photos** for that stack inside. Add **comments** on the ones that need notes.
2. Set sharing so the link can be viewed (or edit) as needed. Paste the **link** into the table above for that stack.
3. Process each photo into a **frame** (30-second fill) and **trading card** using [FRAME_TRADING_CARD_FORMAT.md](FRAME_TRADING_CARD_FORMAT.md). Use [NAMING_AND_PLACES.md](NAMING_AND_PLACES.md) for real names and places.
4. Catalog and archive so every frame is findable by stack, category, character, or story beat.
